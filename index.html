UI Styles Explanation

The following CSS styles are used to create a user interface (UI) for controlling the 3D environment:

  • .ui-container: This class positions the UI container at the bottom left of the screen with a higher stacking order.
  • .controls-box: A central control box for buttons, positioned at the bottom center of the screen with a semi-transparent background and glowing effects.
  • .control-button: Styles for individual buttons, including padding, color, hover effects, and transitions for a responsive feel.
  • Button Colors: Each button has a unique color representing Pikachu's neon theme (e.g., yellow for zoom in, orange for zoom out, etc.).
  • .neon-button: A special button style with a neon glow effect, designed for prominent UI interactions.

These styles ensure that the buttons are visually appealing and easy to interact with, enhancing user experience.
